'''C   
投影时更改屏幕分辨率: 
    查看当前分辨率 ~$ xrand
    查看某分辨率参数设置 ~$cvt 1920 1080
    新建一个Mode ~$xrandr --mode + 上一条命令的参数
    添加一个mode ~$xrandr --newmode 加上设备名 加上分辨率参数 
    删除一二分辨率 ~$xrandr --delmode 外接设备名
    简易方法：直接在用户界面显示设置，设置分辨率

    tips: 外接设备默认为双屏幕，打开镜像可以直接投影
'''
